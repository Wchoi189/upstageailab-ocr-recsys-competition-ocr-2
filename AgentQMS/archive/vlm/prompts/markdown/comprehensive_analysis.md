# Comprehensive Analysis Prompt

Perform a complete analysis combining defect detection, input characteristics, and comparison (if applicable).

## Task

Provide a comprehensive analysis covering all aspects of the image(s) including defects, input characteristics, and any comparisons.

## Analysis Sections

1. **Visual Artifacts** (What does the output look like?)
   - Distortion types
   - Key features
   - Comparison with baseline

2. **Input Characteristics** (What is unique about the source?)
   - ROI coverage
   - Contrast/lighting
   - Geometry

3. **Geometric/Data Analysis** (The Math)
   - Mask topology
   - Corner detection quality
   - Transform matrix assessment

4. **Hypothesis & Action Items**
   - Theory on why issues occurred
   - Proposed fixes
   - Recommendations

## Output Format

Provide a comprehensive structured analysis covering all sections above.
